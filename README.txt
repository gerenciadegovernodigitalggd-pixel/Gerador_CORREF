Gerador de Documentos - CORREF

Como usar:
1. Mantenha index.html, style.css, script.js e a pasta modelos juntos.
2. Para hospedar no GitHub Pages, envie todos esses arquivos extraídos para o repositório.
3. Abra o index.html pelo link do GitHub Pages.
4. Preencha os dados e clique em Gerar DOCX.

Observação:
O gerador usa bibliotecas via CDN (internet): PizZip, Docxtemplater e FileSaver.
