Regras aplicadas

  • Nome completo gera as iniciais automaticamente.
  • Matrícula também gera matrícula mascarada para o extrato.
  • O campo Cargo preenche cargo/origem e cargo do ato decisório.
  • Se a readaptação for definitiva, o prazo e o trecho de reavaliação são removidos do parecer.
  • Se for temporária, o trecho de reavaliação entra com vírgula após “Decreto 17.000”.
  • O campo Determinação é grande para textos longos do ato decisório.

Observação: para funcionar no GitHub Pages, mantenha a pasta modelos no mesmo nível do index.html.
